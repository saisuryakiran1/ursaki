# UrSaKi Watch (Android)

Wear OS stub for Pulse Guardian — HRV via Health Services API.

## Structure

- `app/src/main/java/com/ursaki/watch/` — Kotlin Compose UI + HRVCollector stub

## Requirements

- Android Studio Ladybug+
- Wear OS 5+ emulator or device
- Health Services API permissions
